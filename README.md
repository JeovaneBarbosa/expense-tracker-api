How to run locally on Mac:

        python3 -m venv .venv
        source venv/bin/activate
        pip install -r requirements.txt
        uvicorn nutri_info_api.main:app --reload

How to locally run on Windows:

        python3 -m venv .venv
        source venv\Scripts\activate.bat
        pip install -r requirements.txt
        uvicorn nutri_info_api.main:app --reload

How to run Unit Tests:
